## Requirements

- Windows 10 or Windows 11 (64-bit)
- One or two USB cameras (webcams)
- No internet connection required after install

## Installation

1. Download `RefereePro-Setup-x.x.x.exe` from the link above
2. Run the installer and follow the steps
3. Launch RefereePro from the desktop shortcut or Start Menu

No administrator rights required — installs per user.

## What It Does

Records continuously from one or two cameras simultaneously and lets you instantly review any moment with frame-by-frame scrubbing.

- **Live Delay** — watch footage from a few seconds ago without entering review mode
- **Review** — timeline scrubbing, skip buttons (±0.1s to ±10s), playback speeds (0.25x–4x)
- **Bookmarks** — mark interesting moments during a game and jump back to them instantly
- **Director Mode** — monitor multiple stations from one screen, or from any browser on the network
- **Snapshots** — save PNG images of both cameras at any moment

## Multi-Station Setup

Install RefereePro on multiple laptops. Enable Director Mode on one — all stations appear automatically. No configuration needed.

Open the Director web interface from any browser on the network (tablet, phone, laptop):

```
http://<director-ip>:9080
```
